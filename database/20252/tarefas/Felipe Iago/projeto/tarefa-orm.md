## Resumo sobre ODBC

ODBC (Open Database Connectivity) é uma API padrão para acessar bancos de dados de forma independente da linguagem de programação.  
Em Python, utilizamos o pacote **pyodbc** para:
- Conectar-se a diferentes bancos de dados, incluindo PostgreSQL.  
- Executar comandos SQL diretamente.  
- Recuperar e manipular dados dentro da aplicação Python.  

---

## Resumo sobre ORM

ORM (Object-Relational Mapping) é uma técnica que permite mapear tabelas de um banco de dados como **classes e objetos** na linguagem de programação.  
Em Python, utilizamos o framework **SQLAlchemy**, que permite:
- Criar classes que representam tabelas do banco.  
- Inserir, atualizar, listar e excluir dados sem escrever SQL diretamente.  
- Gerenciar relacionamentos entre tabelas de forma orientada a objetos.
