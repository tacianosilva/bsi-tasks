# Tarefa Individual - ODBC e ORM

## Resumo sobre ODBC (Questão 2.b)

ODBC (Open Database Connectivity) é um padrão que permite que aplicações se conectem a diferentes Sistemas Gerenciadores de Banco de Dados (SGBDs) por meio de uma interface comum. No desenvolvimento em Python, utilizamos bibliotecas de acesso ao banco que permitem executar comandos SQL diretamente, oferecendo controle sobre consultas, transações e operações de inserção, atualização e consulta de dados.

## Resumo sobre ORM (Questão 2.c)

ORM (Object-Relational Mapping) é uma técnica que permite mapear tabelas de um banco de dados para classes da linguagem de programação. Com isso, registros são manipulados como objetos, reduzindo a necessidade de escrever SQL manualmente.

**Framework utilizado:** Django ORM.

O Django ORM facilita operações como inserção, atualização, remoção e consulta de dados, tornando o código mais organizado, legível e de fácil manutenção.
