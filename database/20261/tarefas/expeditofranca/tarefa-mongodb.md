# Resumo MongoDB e MongoDB+Python
O MongoDB é um Sistema Gerenciador de Banco de Dados (SGBD) do tipo NoSQL, amplamente utilizado em aplicações modernas devido à sua flexibilidade, escalabilidade e alto desempenho. Diferente dos bancos relacionais tradicionais, o MongoDB armazena dados em formato de documentos, normalmente utilizando a estrutura BSON (Binary JSON), uma versão binária do JSON. O MongoDB não utiliza tabelas e linhas como bancos relacionais. Em vez disso, os dados são armazenados em coleções (collections) compostas por documentos (documents).

Na linguagem Python, o MongoDB é frequentemente utilizado por meio da biblioteca oficial PyMongo, que permite conectar aplicações Python ao banco de dados de forma simples e eficiente.
